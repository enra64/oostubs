261 /*
262  * The following methods receive and return a reference to the current O_Stream
263  * object. Class O_Stream defines an operator that can be used to call this so
264  * called manipulators. It is even possible to embed the output of the method
265  * into the input of the stream.
266  * Main goal of the manipulators is to influence the display of the following
267  * output (e.g. by choosing a basis for the display of digits).
268  *
269  * To indicate with which base a number is printed,
270  * an indicator prefix should preceed the actual number.
271  *
272  * Prefixes for bases:
273  *
274  * bin: 0b
275  * oct: 0
276  * dec: no prefix
277  * hex: 0x
278  *
279  * The position of a minus sign is before the prefix.
280  *
281  */
